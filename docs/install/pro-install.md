# 企业版 安装

::: tip 提示
企业版安装后试用时长为2个月。
:::

## docker 官方镜像

```shell
curl -sSL https://f.typesafe.cn/next-terminal/docker-compose.yml > docker-compose.yml
docker-compose up -d
```

## 阿里云镜像

```shell
curl -sSL https://f.typesafe.cn/next-terminal/aliyuns/docker-compose.yml > docker-compose.yml
docker-compose up -d
```
